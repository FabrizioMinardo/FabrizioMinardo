Proyectos Destacados

UGAB – Sistema para Restaurante

Plataforma web para un restaurante con 30+ años y hasta 800 comensales por fin de semana. Facilita la atención directa desde la mesa mediante QR, pide la cuenta y deja opiniones, con integración a Telegram para notificaciones instantáneas.

  • Tecnologías: JavaScript, HTML, CSS, Telegram Bot API
  • Rol: Desarrollo fullstack e integración
  • Impacto: Mejoró la atención y redujo tiempos de espera.
  • Beneficio extra: Incrementó la fidelización de clientes y se redujeron gastos operativos.

TuFut

Sistema de gestión para un club que permite administrar reservas, pagos y miembros de manera segura y eficiente. Se eliminaron los errores de duplicación y falta de registros. Se aglizó el proceso de reservas y se automatizó el flujo de trabajo. Se registró y gestionó los pagos de manera correcta.

  • Tecnologías: MySQL, Docker, PHP
  • Rol: Desarrollo backend, administración de base de datos.
  • Impacto: Simplificó la administración del club y mejoró la experiencia del usuario
  • Funciones clave: Se logró una buena administración de reservas y pagos.
